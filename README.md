# 🤖 IA Mentor de Carreira: Descubra Seu Futuro em Tech

Este projeto foi desenvolvido como parte do **Desafio de Projeto da DIO - Inteligência Artificial na Prática**.  
O objetivo é criar um **mentor de carreira com IA**, capaz de orientar pessoas sobre caminhos possíveis dentro da tecnologia.

---

## 🚀 Funcionalidades
- Orientação personalizada de carreira em tecnologia
- Sugestão de trilhas de estudo (frontend, backend, dados, IA, etc.)
- Simulação de entrevistas técnicas
- Análise de perfil com base em habilidades informadas

---

## 🛠️ Tecnologias Utilizadas
- Python 3.10+
- Bibliotecas: `transformers`, `scikit-learn`, `pandas`, `streamlit`
- GitHub Actions para automação de testes e CI/CD

---

## 📂 Estrutura
- `src/` → Código principal da IA
- `data/` → Exemplos de perfis e datasets
- `docs/` → Documentação adicional
- `requirements.txt` → Dependências do projeto
